Run preprocess.py
    -> loads generated and real data
    -> preprocesses them for use in evaluation
    -> saves preprocessed data to disk

Run fidelity.py
    -> loads generated unprocessed data
    -> saves fidelity metrics

Run utility.py
    -> loads preprocessed data
    -> runs utility models 
    -> saves results

Run privacy.py
    -> loads preprocessed data
    -> runs privacy models 
    -> saves results
